1. 처음 다운받을 때

  ① 깃 로그인 (본인아이디 / 이메일 사용)

  ② 다운받고 싶은 폴더로 이동
     - ex) cd c:\git

  ③ 깃 다운받기
    - git clone 주소
    - git clone https://github.com/wlsdn158/project.git

  ④ 다운받고 싶은 폴더안에 프로젝트의 이름으로 폴더가 생성된 후, 그 안에 파일이 다운로드 됩니다.
    - c:\git\project 안에 다운로드

  ⑤ 생성된 폴더안에 .git폴더가 있는지 확인합니다

  ⑥ 끝~

  ★ .git폴더가 중요한 이유
      1. .git폴더안에는 우리가 다운로드한 깃의 주소와 브랜치의 정보들이 저장됩니다.
      2. 따라서 git fetch 명령어를 사용해서 최신화하지 않으면 git을 clone하거나 push할 수 없습니다.


2. 다운로드 한 이후에 작업내용을 최신화하는 방법

  ① 깃 로그인 (본인아이디 / 이메일 사용)
    - git config user.name "아이디"
    - git config user.email "이메일주소"

  ② .git폴더가 있는 디렉토리로 이동
    - ex) cd c:\git\project

  ③ 깃 최신화하기(=동기화)
    - git fetch
    - ★최신화하기전에 반드시!!!! 작업한 파일은 다른곳에 백업해두거나 다른폴더로 옮겨야 합니다★
  
  ④ 사용하고 있는 브렌치 확인 + 존재하는 브렌치 목록 확인
     - git branch -r
       (사용하고 있는 중인 브렌치는 앞에 * 이 붙는다.

  ⑤ 내가 사용할 브랜치 선택
     - git checkout 사용할 브렌치이름 (브렌치 이름 띄어쓰기 불가능)

  ⑥ 다운받은파일과 브랜치안의 파일이 일치하는지 확인합니다.

  ⑦ 끝~
