1. 이미 존재하는 Branch에 업로드

  ① 깃 로그인 (본인아이디 / 이메일 사용)

  ② .git폴더가 있는 디렉토리로 이동
    - ex) cd c:\git\project

  ③ 깃 최신화하기(=동기화) (★필수★)
    - git fetch

  ④ 깃허브에 올릴 파일을 .git폴더가 있는 디렉토리로 복사(반드시 !복사!)
    - ★업로드할 파일을 미리 폴더에 옮긴 후, git fetch 명령어를 입력하면, 업로드할 파일이 사라집니다!!★

  ⑤ 브랜치 목록확인
    - git branch -r

  ⑥ 사용할 브랜치 선택 (예시 : 브랜치 이름이 '20201104' 입니다.)
    - git checkout 20201104

  ⑦ 깃에 업로드할 파일 선택 (예시 : 폴더이름이 '연습'입니다.)
    - git add 연습
    - git add abc.txt (파일 한개씩 업로드할 경우, 해당 파일의 확장자명까지 적어야 합니다)

  ⑧ 업로드할 파일/폴더에 대한 설명을 입력
    - git commit -m "파일에 대한 설명을 써주세요"

  ⑨ 지금까지 입력한 내용들을 깃에 업로드(마지막)
    - git push origin 20201104
    - git push 만 입력해도 됩니다!

  ⑩ 끝~


2. 브랜치를 생성한 후, 업로드

  ② .git폴더가 있는 디렉토리로 이동
    - ex) cd c:\git\project

  ③ 새로운 브렌치를 생성하고, 생성한 브랜치를 사용
     - git checkout -b 사용할 브렌치이름 (브렌치 이름 띄어쓰기 불가능)

  ④ 사용하고 있는 브렌치 확인 + 존재하는 브렌치 목록 확인
     - git branch 
       (사용하고 있는 중인 브렌치는 앞에 * 이 붙는다.

  ⑤ 깃 최신화하기(=동기화) (★필수★)

    - ★업로드할 파일을 미리 폴더에 옮긴 후, git fetch 명령어를 입력하면, 업로드할 파일이 사라집니다!!★

  ⑦ 깃에 업로드할 파일 선택 (예시 : 폴더이름이 '연습'입니다.)

  ⑧ 업로드할 파일/폴더에 대한 설명을 입력
    - git commit -m "파일에 대한 설명을 써주세요"

    - git push origin 20201104

  ⑩ 끝~
